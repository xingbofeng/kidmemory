# Illustrator Visual Personas

These files are not imitation prompts. They are distillations of picture-book visual systems.

Use them as:

1. a visual director's brief
2. a prompt vocabulary source
3. a QA checklist for page consistency
4. a boundary document that prevents direct copying

Default core visual personas:

- `leo-lionni-fable-collage.md`
- `eric-carle-bright-painted-paper.md`
- `beatrix-potter-naturalist-watercolor.md`
- `maurice-sendak-dream-crosshatch.md`
- `tove-jansson-nordic-poetic-line.md`
- `cai-gao-chinese-folk-poetic.md`
