# Print Package Manifest

Expected output structure:

```text
project/
  pages/
    00_cover.png
    01_page.png
    02_page.png
    ...
    09_back_cover.png
  print/
    picturebook_print_single_pages_bleed.pdf
    picturebook_interior_pages_bleed.pdf
    picturebook_cover_front_back_bleed.pdf
    picturebook_combined_proof.pdf
    picturebook_contact_sheet.png
    picturebook_print_package.zip
```

## Recommended Sizes

- Working image ratio: 3:4 vertical
- Common print target: 6x8 inch
- Print resolution: 300 DPI where possible
- Bleed: add if printer requests it

## Handoff Notes

Send printers:

- single-page bleed PDF for all pages, or
- separate cover/front-back PDF and interior PDF if requested

Keep:

- combined proof PDF for review
- contact sheet for order check
- page PNGs for editing or replacement
