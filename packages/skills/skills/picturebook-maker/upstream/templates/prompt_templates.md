# Prompt Templates

## Character Sheet Prompt

```text
Create a character design sheet for a children's picture book protagonist.

Visual system:
[chosen visual system summary]

Character:
[name and role]

Fixed identity:
[face, body, clothing, marks, props]

Show:
- front view
- side view
- 3 emotional expressions
- one prop
- clean warm background

No text, no letters, no watermark.
Do not imitate any named artist. Use the selected visual system as material, composition, and emotional guidance only.
```

## Page Image Prompt

```text
Vertical 3:4 children's picture book illustration.

Visual system:
[material, color, composition, emotional rules from illustrator md]

Character continuity:
[fixed character DNA]

Page:
[page number and title]

Scene action:
[one visible action]

Emotion:
[one emotion]

Objects:
[scene objects]

Composition:
Main subject in upper 70 percent. Leave calmer lower area for later text overlay.

Negative prompt:
No text, no letters, no captions, no watermark.
Do not imitate any named artist. Do not copy known characters or book compositions.
Avoid character drift, clutter, unreadable action, glossy 3D unless the visual system requires it.
```

## Cover Image Prompt

```text
Vertical 3:4 children's picture book cover illustration.

Visual system:
[chosen visual system]

Book idea:
[one-sentence story question]

Cover image action:
[hero image]

Leave clean negative space for title and subtitle.

No text, no letters, no captions, no watermark.
Do not imitate any named artist or copy recognizable book designs.
```
