# QA Checklist

## Per-Page Image QA

- [ ] The page has one clear visible action.
- [ ] The character identity matches the character sheet.
- [ ] The emotional beat is readable.
- [ ] The visual system matches prior pages.
- [ ] There is room for text overlay.
- [ ] No unwanted generated text appears in the image.
- [ ] No copied recognizable character/page composition appears.

## Text Layout QA

- [ ] Chinese text is complete.
- [ ] Title and body are readable.
- [ ] No text overlap.
- [ ] No clipping.
- [ ] Text card does not cover the main action.
- [ ] Page number/order is correct.
- [ ] Layout choice matches this page's action and text length.
- [ ] The book has deliberate layout rhythm, not accidental repetition.

## Book QA

- [ ] Cover title is strong and child-readable.
- [ ] Page sequence reads as a story.
- [ ] Character continuity holds across all pages.
- [ ] Visual rhythm has variety but not drift.
- [ ] Contact sheet shows correct order.
- [ ] PDF opens in system preview.
- [ ] Print files and page PNGs are packaged.

## User Confirmation QA

- [ ] User approved the story question.
- [ ] User approved the story voice.
- [ ] User approved the visual system.
- [ ] User approved the layout rhythm.
- [ ] User approved the page plan.
- [ ] User approved the character sheet.
- [ ] User approved or delegated approval for each generated page.
- [ ] User approved the cover title.
- [ ] User saw the final contact sheet before delivery.
