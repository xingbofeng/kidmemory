# KidMemory Desktop Flutter

macOS Flutter 桌面端。

## Commands

```bash
flutter pub get
flutter analyze
flutter test
flutter run -d macos
```

当前 UI 实现覆盖桌面主流程：

```text
Setup -> Sample Dataset -> Child Profile -> Asset Library -> Generate / Preview / Export
```

桌面视觉按 `docs/images/desktop-*.png` 的温暖卡片、左侧导航、状态卡、素材网格和导出侧栏实现。
