# Claude Code Entry Point

Claude Code should treat `AGENTS.md` as the canonical project instruction file for this repository.

Before starting work, read these files if they exist:

1. `AGENTS.md`
2. `CONTEXT.md`
3. `handoff.md`
4. `implementation-notes.md`

Do not duplicate long-running project rules in this file. Keep shared development workflow, build/test commands, commit conventions, safety boundaries, and Codex/agent workflow guidance in `AGENTS.md`.

Project-specific domain language and architecture context belongs in `CONTEXT.md`.

End-of-session or interrupted-work notes belong in `handoff.md`.
