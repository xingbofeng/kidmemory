对，你理解得对：**如果按“最小改动”方案，页面主体其实不用大改。**

它不是新增一个全新的 onboarding 页面，而是：

> **复用现在的「示例数据集」页面，把它变成欢迎页里“先用示例体验”的落地页。**

也就是说，真正要改的是**流程、状态和按钮行为**，不是大幅重画页面。

---

# 具体流程

## 入口 1：首次没有孩子档案

用户第一次打开 KidMemory，没有孩子档案时，看到当前这个空状态首页：

```txt
还没有孩子档案

[添加孩子档案]
[先用示例体验]
```

两个按钮：

```txt
添加孩子档案
  → 打开添加孩子表单

先用示例体验
  → 跳转到现有「示例数据集」页面
```

这里不直接导入示例数据。

---

# 示例体验完整流程

## Step 1：点击「先用示例体验」

跳转到现有页面：

```txt
示例数据集页面
```

或者页面标题可以轻微改成：

```txt
体验示例
```

但这不是必须。
最小改动下，保留现在页面结构也可以。

---

## Step 2：进入示例页后先判断状态

页面加载时检查：

```txt
示例数据是否已经导入
```

状态有四种：

```txt
not_imported  未导入
importing     导入中
imported      已导入
failed        导入失败
```

---

# 状态 1：未导入

页面展示当前这些内容：

```txt
使用隐私安全的虚拟数据集
素材总数
儿童画
手工作品
照片
标签
素材预览
隐私安全说明
```

右侧按钮：

```txt
[导入示例数据集]
[查看示例 PDF]
```

如果示例 PDF 需要导入后才能查看，那就禁用：

```txt
[查看示例 PDF] disabled
说明：导入后可查看示例 PDF
```

用户点击：

```txt
导入示例数据集
```

---

# 状态 2：导入中

点击导入后，页面不跳转，直接进入 loading 状态：

```txt
正在导入示例数据...
```

可以显示简单步骤：

```txt
✓ 创建示例孩子档案
→ 导入示例素材
○ 写入标签
○ 完成
```

按钮禁用：

```txt
[导入中...]
```

防止重复点击。

---

# 状态 3：导入成功

导入完成后，页面右侧状态变成：

```txt
示例数据已导入
```

不要再把「已导入示例数据集」做成大绿色主按钮。

更合理的是：

```txt
示例数据已导入
你可以继续浏览示例素材，或体验生成流程。
```

然后显示操作按钮：

```txt
[浏览示例素材]
[生成示例绘本]
[查看示例 PDF]
[重置数据]
```

如果你想继续最小改动，只保留已有功能，也可以是：

```txt
[查看示例 PDF]
[重置数据]
```

但我建议至少加一个：

```txt
浏览示例素材
```

这个不需要新接口，只是跳转到素材库并切换到示例孩子。

---

# 状态 4：导入失败

如果导入失败，右侧显示：

```txt
导入失败
请检查本地数据库和示例素材文件。
```

按钮：

```txt
[重试导入]
[查看日志]
```

如果暂时没有日志页面，先只做：

```txt
[重试导入]
```

---

# 导入成功后跳转逻辑

## 浏览示例素材

点击：

```txt
浏览示例素材
```

跳转到：

```txt
素材库
```

并设置当前孩子为：

```txt
示例孩子
```

素材库展示示例素材。

如果当前素材库没有 `source=sample` 过滤能力，就先只切换 childId 即可。

---

## 生成示例绘本

点击：

```txt
生成示例绘本
```

跳转到：

```txt
创作台
```

并预填：

```txt
创作类型：儿童绘本
当前孩子：示例孩子
素材来源：示例素材
模板：温暖童趣
页面尺寸：A4 竖版
```

然后用户自己点击：

```txt
开始生成
```

不要在示例页直接自动生成，避免流程不可控。

---

## 查看示例 PDF

点击：

```txt
查看示例 PDF
```

如果已经有静态示例 PDF：

```txt
打开示例 PDF
```

如果没有生成过：

```txt
提示：导入并生成示例绘本后可查看 PDF
```

或者保留当前已有逻辑。

---

## 重置数据

点击：

```txt
重置数据
```

弹确认：

```txt
确定要重置示例数据吗？
这会删除当前示例档案和示例素材，并重新恢复到初始状态。
```

确认后：

```txt
调用现有 reset sample dataset 逻辑
```

完成后回到：

```txt
not_imported 或 imported
```

取决于你的 reset 是“清空”还是“恢复默认”。
我建议 reset 表示：

```txt
删除后重新导入一份干净示例数据
```

如果是删除，则按钮文案应该叫：

```txt
删除示例数据
```

# 最小改动的数据流

```txt
Welcome / Empty Child Page
  ↓ 点击“先用示例体验”
SampleDatasetPage
  ↓ 查询示例数据状态
not_imported
  ↓ 点击“导入示例数据集”
importing
  ↓ 导入成功
imported
  ↓ 用户选择
  ├─ 浏览示例素材 → 素材库
  ├─ 生成示例绘本 → 创作台
  ├─ 查看示例 PDF → PDF 预览
  └─ 重置数据 → reset sample
```

# 接口上需要新增吗

按最小改动：**不需要新增接口**。

只复用已有能力：

```txt
导入示例数据集
重置数据
查看示例 PDF
读取当前 dataset / child / asset 状态
```

如果现在没有“示例数据是否已导入”的明确接口，也不用马上新增，可以用现有数据推断：

```txt
查是否存在 sample child
或者查是否存在 sample source assets
或者查 sample dataset import report
```

后续再补一个正式状态接口：

```txt
GET /sample-dataset/status
```

但最小改动可以先不加。

# 最小改动提示词

````md
# 任务：实现“先用示例体验”的最小流程

请不要新增完整新页面，不要新增后端接口，不要大改 UI。基于现有欢迎空状态页和现有「示例数据集」页面实现最小流程。

## 目标

当用户还没有孩子档案时，可以点击「先用示例体验」，进入现有示例数据集页面，手动导入示例数据，并继续浏览示例素材或体验生成流程。

## 需要改动

### 1. 欢迎空状态页

在「添加孩子档案」按钮下方新增一个次级按钮：

```txt
先用示例体验
````

点击后跳转到现有「示例数据集」页面。

注意：点击该按钮不要直接导入示例数据。

### 2. 示例数据集页面

保留现有页面结构、接口和功能，只调整文案和状态。

页面副标题改为：

```txt
使用隐私安全的虚拟素材，快速体验 KidMemory 的素材库、创作台和导出流程。
```

主说明卡文案改为：

```txt
使用隐私安全的虚拟素材
所有示例内容均为虚拟素材，仅用于功能演示与体验，不包含任何真实儿童数据。
```

### 3. 示例数据状态

页面需要根据示例数据是否已导入显示状态：

```txt
not_imported
importing
imported
failed
```

如果暂时没有状态接口，可以通过是否存在示例孩子或示例素材来推断。

### 4. 未导入状态

右侧显示：

```txt
状态：未导入
[导入示例数据集]
[查看示例 PDF]
```

如果 PDF 需要导入后才能看，则禁用查看 PDF。

### 5. 导入中状态

点击导入后显示：

```txt
正在导入示例数据...
```

按钮 disabled，避免重复点击。

### 6. 已导入状态

不要把「已导入示例数据集」显示成大号绿色主按钮。

改成状态提示：

```txt
示例数据已导入
你可以继续浏览示例素材，或体验生成流程。
```

按钮显示：

```txt
[浏览示例素材]
[生成示例绘本]
[查看示例 PDF]
[重置数据]
```

如果暂时没有“生成示例绘本”的预填逻辑，可以先隐藏这个按钮，只保留：

```txt
[浏览示例素材]
[查看示例 PDF]
[重置数据]
```

### 7. 浏览示例素材

点击后跳转到素材库，并切换到示例孩子或示例素材范围。

不要新增接口，优先复用现有素材库过滤和当前孩子选择逻辑。

### 8. 生成示例绘本

如果已有创作台路由，则点击后跳转到创作台，并预填：

```txt
创作类型：儿童绘本
孩子：示例孩子
模板：温暖童趣
```

不要自动开始生成。

### 9. 重置数据

保留现有重置逻辑。

点击前弹确认：

```txt
确定要重置示例数据吗？
```

### 10. 不要做

* 不要新增新页面
* 不要新增后端接口
* 不要自动导入示例数据
* 不要改真实孩子数据逻辑
* 不要让示例数据和真实数据混在一起
* 不要显示 sample-child-001 给普通用户
* 不要显示 cld003-local / MCP Test 等技术名

## 验收标准

* [ ] 欢迎页有「先用示例体验」按钮。
* [ ] 点击后跳转到现有示例数据集页面。
* [ ] 点击后不会自动导入数据。
* [ ] 用户可以在示例页手动导入示例数据。
* [ ] 导入中按钮 disabled。
* [ ] 已导入后显示状态，不再显示成大号绿色按钮。
* [ ] 已导入后可以浏览示例素材。
* [ ] 已导入后可以查看示例 PDF。
* [ ] 可以重置示例数据。
* [ ] 不新增后端接口。
* [ ] 不大改页面结构。

